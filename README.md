## What is this?

A self-contained, zero-dependency Snake game disguised as a single line of JavaScript. Copy it, paste it into your browser's DevTools console, and a floating 🐍 button appears on any webpage — click it to play.

## Features

- **One-liner injection** — copy, paste, play. No files, no setup, no extensions.
- **Floating & draggable** — the 🐍 button gently bobs in the corner. Drag it anywhere you want.
- **Full game in a modal** — frosted-glass overlay with retro neon aesthetics.
- **Pause & resume** — Space to toggle pause, Escape to close.
- **Game over screen** — shows your score with a one-click restart.
- **Non-destructive** — won't interfere with the underlying page. Guard against double-injection.

## How to use

1. Open **any** webpage
2. Open DevTools (`F12` or `Ctrl+Shift+J` / `Cmd+Option+J`)
3. Go to the **Console** tab
4. Paste the [one-liner](#the-code) and hit `Enter`
5. Click the 🐍 button that appears

## Controls

| Key | Action |
|---|---|
| `W` / `↑` | Move up |
| `S` / `↓` | Move down |
| `A` / `←` | Move left |
| `D` / `→` | Move right |
| `Space` | Pause / Resume |
| `Esc` | Close game |
